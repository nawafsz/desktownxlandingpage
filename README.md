# مشروع DeskTown - صفحة الهبوط

هذا المشروع عبارة عن صفحة هبوط حديثة ومتجاوبة لخدمة DeskTown للمكاتب السحابية.

## محتويات المشروع
- **index.html**: الصفحة الرئيسية (الهيكل والمحتوى).
- **css/style.css**: ملف التنسيقات (الألوان، الخلفيات، الخطوط، والتجاوب مع الجوال).
- **js/script.js**: ملف الجافاسكريبت (التفاعلات البسيطة والتحركات).
- **images/**: مجلد الصور (يحتوي على الخلفية وصور الخدمات).

## كيفية التشغيل محلياً (على جهازك)
1. قم بفك الضغط عن المجلد (إذا كان مضغوطاً).
2. افتح ملف `index.html` باستخدام أي متصفح (Chrome, Edge, Firefox).
3. ستظهر الصفحة وتعمل بشكل كامل.

## كيفية الرفع على GitHub (الطريقة اليدوية)

لرفع المشروع لكي تتمكن من استضافته لاحقاً (على Render أو غيره):

1. اذهب إلى [GitHub.com](https://github.com) وسجل الدخول.
2. اضغط على علامة **(+)** في الزاوية العلوية واختار **New repository**.
3. اكتب اسم المستودع (مثلاً `desktown-landing`) واضغط **Create repository**.
4. في الصفحة التالية، اضغط على الرابط **uploading an existing file**.
5. قم بتحديد **جميع الملفات والمجلدات** في مجلد المشروع الخاص بك (`index.html`, مجلد `css`, مجلد `js`, مجلد `images`).
6. اسحبهم وأفلتهم في المتصفح.
7. انتظر حتى تنتهي عملية الرفع، ثم اكتب رسالة في خانة "Commit changes" (مثلاً: "Initial commit") واضغط الزر الأخضر **Commit changes**.

## كيفية الرفع على Render (للنشر)

1. اذهب إلى [Render.com](https://render.com) وسجل حساباً جديداً (يمكنك استخدام حساب GitHub).
2. اضغط على **New +** ثم اختر **Static Site**.
3. اربط حسابك بـ GitHub إذا لم تكن قد فعلت ذلك، وستظهر لك قائمة بمستودعاتك.
4. اختر مستودع `desktown-landing` الذي رفعته للتو.
5. اضغط **Connect**.
6. ستظهر لك إعدادات النشر:
    - **Environment Variables**: اتركها فارغة (لا نحتاج إليها لهذا الموقع).
    - باقي الإعدادات: اتركها كما هي.
7. اضغط **Create Static Site**.
8. انتظر دقيقة وسيتم نشر موقعك وسيعطيك رابطاً (مثلاً: `https://desktown-landing.onrender.com`).

## تعديل الصور والنصوص
- **النصوص**: يمكنك تعديل أي نص مباشرة بفتح ملف `index.html` باستخدام أي محرر نصوص (مثل Notepad أو VS Code).
- **الصور**:
    - الصور موجودة في مجلد `images`.
    - لتغيير الخلفية، استبدل ملف `background.jpg` أو عدل الرابط في `css/style.css`.
    - لتغيير صور الخدمات، استبدل الصور في المجلد بنفس الأسماء، أو عدل الأسماء في `index.html`.

## تفعيل نموذج التسجيل (Formspree)
لجعل النموذج يعمل ويرسل البيانات إلى بريدك الإلكتروني الحقيقي:
1. اذهب إلى موقع [Formspree.io](https://formspree.io/) وأنشئ حساباً مجانياً.
2. أنشئ نموذجاً جديداً (New Form) وضع بريدك الإلكتروني كوجهة.
3. سيعطيك الموقع رابطاً خاصاً (مثلاً: `https://formspree.io/f/xnqoykxz`).
4. افتح ملف `index.html` وابحث عن السطر الذي يبدأ بـ `<form action=...`.
5. استبدل الرابط القديم بالرابط الجديد الخاص بك.
6. الآن سيعمل النموذج وسيرسل البيانات إلى بريدك مباشرة.
